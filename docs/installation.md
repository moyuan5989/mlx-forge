# Installation

## Quick Install

```bash
pip install mlx-forge
```

## Development Setup

```bash
git clone https://github.com/moyuan5989/mlx-forge.git
cd mlx-forge
pip install -e ".[dev]"
```

## Requirements

- **Hardware**: Apple Silicon Mac (M1, M2, M3, M4 series)
- **OS**: macOS 13+ (Ventura or later)
- **Python**: 3.10, 3.11, or 3.12
- **MLX**: 0.18.0+

## Optional Dependencies

### WandB Integration

```bash
pip install mlx-forge[wandb]
```

### Documentation (for contributors)

```bash
pip install mkdocs mkdocs-material
```

## Verify Installation

```bash
mlx-forge --version
```
